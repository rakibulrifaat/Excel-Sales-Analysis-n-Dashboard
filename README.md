# Excel-Sales-Analysis-n-Dashboard
Coffee Sales Analysis and interactive Dashboard built in Excel — end-to-end project with data cleaning, lookups, pivot tables, slicers, and dynamic visuals.
## Project Overview
This project transforms raw sales data into actionable business insights:
- Cleaned and combined data from multiple sheets
- Used lookups (XLOOKUP + INDEX-MATCH) to enrich orders
- Created dynamic visuals with PivotTables, PivotCharts, Slicers, and Timeline
- Key metrics: Total sales over time, sales by country, top 5 customers, coffee type performance
 ## Tools & Features Used
- **Excel Formulas**: XLOOKUP, INDEX-MATCH, IF, multiplication for sales calculation
- **Data Preparation**: Remove duplicates, date/number formatting, convert to Table
- **Analysis**: Pivot Tables & Charts, Slicers (Roast Type, Size, Loyalty Card), Timeline filter
- **Dashboard**: Interactive filtering, clean layout with custom colors

1. Download `Coffee Business Analysis.xlsx`

3. Use the Timeline and Slicers on the Dashboard sheet to filter data
4. Explore formulas and Pivot setup on other sheets
